# NIS2 Asset Management Database

A relational database developed in PostgreSQL to manage IT assets, digital services, dependencies and organizational responsibilities according to the NIS2 Directive and the requirements of the Italian National Cybersecurity Agency (ACN).

---

## Project Overview

This project was developed as my Bachelor's final project in Computer Science.

The objective is to design and implement a relational database that supports the collection, organization and reporting of information required for NIS2 compliance.

The database allows organizations to manage:

- IT Assets
- Digital Services
- Dependencies between assets and services
- Organizational responsibilities
- Criticality classifications

---

## Features

- Relational database design
- Entity-Relationship (ER) modeling
- Database normalization (up to Third Normal Form)
- PostgreSQL implementation
- SQL reporting queries
- SQL Views
- CSV export for reporting
- Asset dependency management
